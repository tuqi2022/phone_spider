# 快速开始指南

## 🚀 3步开始使用

### 1. 激活虚拟环境

```bash
cd /home/nstar/data/data/code/numb_check/telecom_spider
source venv/bin/activate
```

### 2. 运行爬虫

#### 单城市版本
```bash
# 爬取深圳地区
python spider_simple.py

# 爬取广州地区
python spider_simple.py --city 广州
```

#### 多城市版本（推荐）✨
```bash
# 爬取多个城市
python spider_multi_city.py --cities 深圳 广州 东莞

# 或者只爬取一个城市
python spider_multi_city.py --cities 深圳
```

### 3. 查看结果

爬取完成后，会在当前目录生成 JSON 文件：

- 单城市：`phones_深圳_20260107_120341.json`
- 多城市：`phones_multi_20260107_120341.json`

查看内容：

```bash
cat phones_*.json
```

## ⚙️ 自定义搜索模式

如果你想搜索特定的号码模式（不只是 000* 到 999*），可以修改 `spider_simple.py` 中的循环：

```python
# 原代码（第56行附近）：
for i in range(10):
    pattern = f'{i}{i}{i}*'

# 修改为自定义模式：
patterns = ['520*', '1314*', '888*', '666*', '168*']
for pattern in patterns:
    print(f'\n正在搜索模式: {pattern}')
    # ... 其余代码保持不变
```

## 🐛 常见问题

### 问题1: 找不到 python 命令
```bash
# 使用 python3
python3 spider_simple.py
```

### 问题2: 某些模式没有找到号码
这是正常的，有些模式确实没有可用号码。

### 问题3: 爬取速度慢
这是为了避免对服务器造成压力，默认每次搜索间隔 3 秒。

## 📊 性能说明

- 搜索 10 个模式大约需要 1-2 分钟
- 每个模式平均可找到 0-10 个号码
- 总共可能找到 10-50 个号码（取决于可用号码数量）
